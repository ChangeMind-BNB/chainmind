Quick Start

Register

curl -X POST https://api.chainmind.xyz/v1/register

Claude Code

export ANTHROPIC_BASE_URL=https://api.chainmind.xyz/proxy/{YOUR_TOKEN}
export ANTHROPIC_API_KEY=chainmind
claude "Your prompt"

Python (Anthropic SDK)

import anthropic

client = anthropic.Anthropic(
    base_url="https://api.chainmind.xyz/proxy/{YOUR_TOKEN}",
    api_key="chainmind",
)

response = client.messages.create(
    model="claude-sonnet-4-6",
    max_tokens=1024,
    messages=[{"role": "user", "content": "Hello"}],
)

Python (OpenAI SDK)

from openai import OpenAI

client = OpenAI(
    base_url="https://api.chainmind.xyz/proxy/{YOUR_TOKEN}/v1",
    api_key="chainmind",
)

response = client.chat.completions.create(
    model="gpt-4o",
    messages=[{"role": "user", "content": "Hello"}],
)

Smart Model Routing

curl -X POST https://api.chainmind.xyz/proxy/{TOKEN}/v1/messages \
  -H "X-ChainMind-Budget: 500000" \
  -H "X-ChainMind-Route: auto" \
  -H "X-ChainMind-Min-Tier: mid" \
  -H "Content-Type: application/json" \
  -d '{"max_tokens": 1024, "messages": [{"role": "user", "content": "Hello"}]}'

# Response includes: X-ChainMind-Model-Used header

Smart Model Routing

Set a budget cap per request. ChainMind picks the cheapest model that can handle it.

X-ChainMind-Budget: 500000

Budget in micro-USDC for this request (500000 = $0.50)

X-ChainMind-Route: auto

Opt-in to smart routing. Without this, the request uses the specified model.

X-ChainMind-Min-Tier: mid

Quality floor: low = any model, mid = skip budget models, high = premium only

Model Priority (cheapest first)

  1. gpt-4o-mini low
  2. grok-4-1-fast low
  3. claude-haiku-4-5 low
  4. qwen3-max mid
  5. gemini-3.1-pro mid
  6. gpt-4o high
  7. claude-sonnet-4-6 high
  8. claude-opus-4-6 high
